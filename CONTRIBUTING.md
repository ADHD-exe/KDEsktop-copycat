# Contributing

Thanks for wanting to contribute! Here's the quick version:

## Setup

```bash
git clone https://github.com/YOUR_USERNAME/KDEsktop-copycat.git
cd KDEsktop-copycat
cargo build
cargo test
```

## Development

- Use `cargo fmt` for formatting
- Use `cargo clippy` for linting  
- Write tests for new features
- Keep commits focused and descriptive

## Areas I want help with

- **Parser**: Handle weird/malformed Plasma configs better
- **TUI**: More navigation options, better visual design
- **Export**: Support more restoration methods
- **KWin**: Better effect/script detection
- **Tests**: Always need more test coverage

## Submitting

1. Fork and branch: `git checkout -b feature/your-thing`
2. Make your changes
3. Test with your actual Plasma setup if possible
4. Push and open PR

That's it. No bureaucracy, just good code.
