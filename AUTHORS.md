## How to Add Yourself

If you've contributed to this project, feel free to add yourself to this list:

1. Fork the repository
2. Edit this `AUTHORS.md` file
3. Add your name and a brief description of your contributions
4. Submit a pull request

Format:
```markdown
- **[Your Name]** - *Your contributions and areas of focus*
```

Examples:
```markdown
- **Jane Doe** - *TUI improvements and bug fixes*
- **John Smith** - *Documentation and examples*
- **Alex Johnson** - *KWin integration and parser enhancements*
```
